Parkinson's Disease Progression Prediction

Python TensorFlow Scikit-learn XGBoost
View Code

Project Overview

Developed a comprehensive machine learning solution to predict Parkinson's disease progression using protein and peptide biomarkers. The system provides both continuous UPDRS scores and severity categories to support clinical decision-making.

Key Features

  • Multi-output neural network (128-64-32 architecture) with batch normalization and dropout
  • 16.6% R² score improvement (3x over baseline) in predicting all 4 UPDRS progression metrics
  • Complementary Random Forest classifiers (F1=0.89 for UPDRS-1) for severity categorization
  • Reduced MAE from 5.16 (baseline) to 4.64 through strategic feature engineering

Technical Details

  • Model Architecture: 3-layer neural network with batch normalization and 20% dropout for regularization
  • Training: Adam optimizer with MSE loss, 50 epochs with early stopping
  • Key Findings: Medication status was the most important feature (35.2% importance)
  • Data Processing: Strategic binning to address class imbalance in severity categories

Impact

This solution provides clinicians with both continuous progression scores and severity categories, enabling more personalized treatment plans. The model's ability to identify key biomarkers contributes to earlier intervention strategies.

Project Overview

Developed a model-agnostic pipeline for text-to-SQL generation using compact transformers (60M-139M parameters) on the Spider benchmark. The system translates natural language questions into SQL queries while considering database schema constraints.

Key Features

  • Fine-tuned T5-Small achieved 27.8% Logical Form Accuracy (LFAcc)
  • Outperformed BART-Small (23.98%) and GPT-2 (20.1%) through schema-aware formatting
  • Implemented constrained decoding to ensure syntactically valid SQL
  • Modular pipeline supports future enhancements with alternative base models

Technical Details

  • Models Evaluated: T5-Small, BART-Small, and GPT-2 on Spider dataset
  • Evaluation Metrics: LFAcc (Logical Form Accuracy), BLEU, Exact Match
  • Key Innovation: Schema-aware input formatting for improved performance
  • Performance: Demonstrated encoder-decoder models' superiority in structured query generation

Impact

This research provides valuable insights into implementing text-to-SQL systems in resource-constrained environments. The findings help bridge the gap between natural language interfaces and database systems, making data more accessible to non-technical users.

Youth Substance Use Determinants (Canada)

Bayesian Stats PyMC3 Lasso Multilevel Models PSM
View Code

Project Overview

Comprehensive statistical analysis of 2018/2019 CSTADS data (60K+ Canadian students) to identify key determinants of youth substance use. Employed multiple modeling approaches to uncover relationships between social, demographic factors and substance use patterns.

Key Features

  • Bayesian Logistic Regression (WAIC: -13901.97) identified grade level and income as key predictors
  • Lasso Regression achieved 0.7779 AUC for alcohol use prediction
  • Multilevel Modeling revealed peer influence as dominant factor (OR=951.35, p<0.001)
  • Propensity Score Matching quantified +0.52 smoking risk from social exposure

Technical Details

  • Data Source: Canadian Student Tobacco, Alcohol and Drugs Survey (CSTADS)
  • Key Finding: 99.98% of smokers reported peer influence vs 54.92% non-smokers (χ²=187.29, p<0.001)
  • Random Forest: Ranked grade level (importance=0.41) as top predictor
  • Methodologies: Bayesian modeling, Lasso regression, Multilevel modeling, PSM

Impact

This analysis provides evidence-based insights for public health policymakers to design targeted substance use prevention programs. The strong association between peer influence and substance use suggests that interventions should focus on social environments.
